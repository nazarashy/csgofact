# CS:GO Server Website Template

A professional, responsive website template designed specifically for a Counter-Strike: Global Offensive (CS:GO) gaming server.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Clean design with CS:GO-themed colors (red, blue accents)
- **Complete Sections**:
  - Hero section with server connection info
  - About section highlighting server features
  - Server information with IPs and status
  - Server rules and punishment system
  - Contact form with multiple contact methods
  - Footer with quick links
- **Interactive Elements**:
  - Smooth scrolling navigation
  - Server IP copy functionality
  - Form validation
  - Animated elements on scroll
  - Dynamic server status simulation

## Files Included

- `index.html` - Main HTML structure
- `styles.css` - All styling and responsive design
- `script.js` - Interactive JavaScript functionality
- `README.md` - This documentation file

## How to Customize

1. **Server Information**:
   - Update the server IP addresses in both HTML and JavaScript
   - Modify server names and descriptions
   - Adjust player counts and server status

2. **Content**:
   - Change the server rules to match your specific ruleset
   - Update contact information (Discord, email, Steam group)
   - Modify the "About" section to describe your server

3. **Styling**:
   - Adjust colors by modifying the CSS variables
   - Change the hero background image URL
   - Modify fonts if needed

4. **Functionality**:
   - Connect the contact form to your backend
   - Integrate with a real server status API instead of the simulated version

## Setup Instructions

1. Upload all files to your web server
2. Update the content with your specific server information
3. Connect the contact form to your preferred method (email service, Discord webhook, etc.)
4. Test all functionality on different devices
